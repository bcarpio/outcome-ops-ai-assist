# Lambda: Generate Code

The generate-code Lambda orchestrates AI-powered code generation from Jira user stories or GitHub issues. It receives triggers from Jira Automation or GitHub webhooks, creates execution plans using Claude via Bedrock with knowledge base context, generates code step-by-step, and opens Pull Requests with the results.

## Key Features

- Dual trigger support: Jira Automation (via SSM) and GitHub webhooks (via API Gateway)
- Async processing via SQS FIFO to avoid Lambda timeout constraints
- Multi-step execution plans generated by Claude with knowledge base context
- Automatic git branch creation, code commits, and Pull Request creation
- Language-agnostic architecture with pluggable backends (Python, TypeScript, Java)
- ADR-driven code generation that follows your documented standards
- Repository configuration via `.outcomeops.yaml` for project-specific context
- Built-in retry and error handling with DLQ support
