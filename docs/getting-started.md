# Getting Started

This guide walks you through the initial setup of OutcomeOps AI Assist in your AWS account. It covers prerequisites, GitHub token configuration, AWS infrastructure provisioning with Terraform, knowledge base population, and verification of the deployed system.

## Key Features

- Step-by-step setup from zero to a working deployment
- GitHub Personal Access Token (classic or fine-grained) configuration
- Terraform-based infrastructure provisioning with environment-specific variables
- Knowledge base initialization with ADRs and code maps
- CLI verification to confirm end-to-end functionality
- Chat UI deployment with Azure AD authentication
- Multi-workspace setup for managing multiple repositories
