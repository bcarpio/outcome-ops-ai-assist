# Example Code Generation Issue

This document provides a reference template for structuring code generation requests as GitHub or Jira issues. It demonstrates how to write a well-formed issue that the AI code generation pipeline can use to produce a Lambda handler, including the user story, handler specification, request/response payloads, test scenarios, and expected file outputs.

## Key Features

- Complete example of a code generation issue with user story format
- Handler specification with trigger type, timeout, and memory configuration
- Request and response payload schemas with validation rules and status codes
- Business logic description with error handling and edge cases
- Comprehensive test scenarios covering success, error, and edge cases
- Expected file list for both new files and modifications to existing infrastructure
- Terraform requirements following existing project conventions
