# Generate Code: TypeScript Backend

The TypeScript backend handles AI-powered code generation for TypeScript and JavaScript applications of any type, including Lambda, Express, Next.js, React, NestJS, and any other Node.js framework. It is language-focused rather than framework-focused, with framework-specific patterns driven by your repository's ADRs and knowledge base.

## Key Features

- Language-focused design that works across all TypeScript/Node.js frameworks
- ADR-driven code generation for framework-specific conventions
- File validation for TypeScript syntax and package.json structure
- Support for Jest and Vitest test pattern generation
- Dependency management across npm, yarn, and pnpm
- Code map discovery of existing project patterns and structures
- Configurable directory pattern recognition (handlers, routes, components, schemas)
- Project context via `.outcomeops.yaml` for project-specific structure
