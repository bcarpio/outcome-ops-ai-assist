# Generate Code: Java Backend

The Java backend handles AI-powered code generation for Java and Kotlin applications of any type, including Spring Boot, Lambda, Quarkus, Micronaut, and any other Java framework. It is language-focused rather than framework-focused, with framework-specific patterns driven by your repository's ADRs and knowledge base.

## Key Features

- Language-focused design that works across all Java/Kotlin frameworks
- ADR-driven code generation for framework-specific conventions
- File validation for Java syntax, pom.xml, and build.gradle
- JUnit 5 test pattern generation for unit, integration, and e2e tests
- Dependency management across Maven and Gradle
- Code map discovery of existing project patterns and package structures
- Standard Maven/Gradle layout support with configurable source roots
- Project context via `.outcomeops.yaml` for project-specific structure
